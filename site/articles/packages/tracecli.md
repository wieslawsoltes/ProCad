# Trace CLI

`ProCad.TraceCli` is an executable project rather than a library package.

It depends on the same core, IO, rendering, and app services used by the desktop application and exposes a repeatable command-line harness for:

- render scene build timing
- frame render timing
- multi-file sweeps
- visual style comparisons
- PNG output generation
- `dotnet-trace` collection

See [Trace CLI](../getting-started/trace-cli.md) for command examples.
