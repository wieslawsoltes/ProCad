  <h1>ProCad</h1>

  <p class="lead"><strong>ProCad</strong> is a .NET CAD inspection, rendering, editing, scripting, collaboration, and reusable-control stack for DWG and DXF documents. It is built around Avalonia, ReactiveUI, Dock, ProDataGrid, AvaloniaEdit, ACadSharp, SkiaSharp, and Lunet-generated documentation.</p>
